User Stories

US-001 Support Analyst

As a support analyst,

I want support requests to be classified automatically,

so that I spend less time on manual triage.

⸻

US-002 Support Analyst

As a support analyst,

I want knowledge articles to be selected automatically,

so that I can begin troubleshooting immediately.

⸻

US-003 Support Analyst

As a support analyst,

I want Jira tickets to contain recommended resolution steps,

so that I can resolve issues faster and more consistently.

⸻

US-004 Support Manager

As a support manager,

I want ticket categorization to be standardized,

so that reporting and workload distribution become more reliable.

⸻

US-005 Knowledge Manager

As a knowledge manager,

I want knowledge articles to be maintained through GitHub,

so that updates can be performed without changing the workflow.

⸻

US-006 End User

As an end user,

I want my support request to reach the correct support team quickly,

so that my issue is resolved faster.

⸻

US-007 Solution Owner

As a solution owner,

I want the workflow to continue operating when AI responses fail,

so that support operations are not interrupted.
